TENTE IMPRIMIR ESSA PAGINA PARA ENTENDER

Foi feito dois estilos CSS: Uma para impressão e outra como página internet mesmo. O objetivo do Media Query é diferenciar a leitura e adaptação de página para cada finalidade e tipo de tela onde o conteúdo será consumido. O media query, nesse caso, foi o comando "Media" colocado no link css, optando entre media impressa e screen.
